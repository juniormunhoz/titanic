Sistema modular em Python para importar, visualizar e realizar análises estatísticas sobre a lista de passageiros do Titanic contida no arquivo train.csv.

- main.py: Ponto de entrada que gerencia a interação com o usuário.
- importacao.py: Funções para carregar o arquivo CSV.
- visualizacao.py: Funções para exibir os dados dos passageiros.
- analise_titanic.py: Cálculos estatísticos e filtragens.
